HTML Host

使い方・制限事項
  • HTMLアップロード: ファイル選択またはドラッグ&ドロップでHTMLファイルをアップロードできます。
  • Driveから読み込む: 対象の共有Driveフォルダに直接HTMLファイルを置き、「スキャン」ボタンを押すと未登録ファイルが自動で読み込まれます。投稿者はファイル所有者が自動設定されます。
  • バージョン管理: 同名ファイルを再アップロードすると、旧バージョンが自動保存されます。「履歴」から過去バージョンの閲覧・復元が可能です。
  • プレビュー: 一覧の「プレビュー」ボタンでページ内容を確認できます。
  • メモ: 各ページに説明メモを追加・編集できます。CLIデプロイ時はHTMLの内容から概要が自動生成されます。
  • CLI デプロイ: Claude Code の /deploy-html スキルでコマンドラインからもデプロイ可能です。メモ・投稿者が自動登録されます。
Claude Code でスキル化する方法
  1. GitHub リポジトリ osamu-araki/gas-html-host をクローン（git clone https://github.com/osamu-araki/gas-html-host.git）
  2. skills/deploy-html-example.md をコピーして ~/.claude/skills/deploy-html/SKILL.md に配置
  3. SKILL.md 内の PROJECT_DIR、FOLDER_ID、WEB_APP_URL を自分の環境に合わせて書き換え
  4. Claude Code で /deploy-html と入力すれば準備完了
スキル化でできること:
  • /deploy-html file.html — HTMLファイルをワンコマンドでデプロイ、公開URLを即取得
  • /deploy-html --list — デプロイ済みページの一覧をターミナルで確認
  • /deploy-html --delete ページ名 — 不要なページをコマンドラインから削除
  • Claude Code が生成したHTML（レポート・ダッシュボード等）を会話中にそのままデプロイ可能
制限事項: ファイルサイズ上限 約500KB/ページ ｜ カスタムドメイン非対応 ｜ CDNからのCSS/JS読み込みは可能 ｜ 閲覧にはGoogle Workspace認証が必要
